# 📇 Agenda de Contatos em Python

Aplicação de linha de comando (CLI) desenvolvida em Python para gerenciamento de contatos.  
O sistema implementa operações completas de CRUD, incluindo marcação de favoritos.

---

## 🚀 Funcionalidades

- ✅ Adicionar novos contatos
- 📋 Listar todos os contatos
- ✏️ Editar informações de um contato
- ⭐ Marcar / desmarcar como favorito
- 🔍 Listar apenas contatos favoritos
- 🗑️ Remover contatos

---

## 🛠️ Tecnologias utilizadas

- Python 3.x
- Estruturas de dados nativas (Listas e Dicionários)

---

## 🧠 Conceitos aplicados

- CRUD (Create, Read, Update, Delete)
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if/else`)
- Manipulação de listas e dicionários
- Tratamento de erros com `try/except`
